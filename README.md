## Building and running

### Local

Install dependencies:

```
npm install
```

Run locally:

```
npm run dev -- --open
```

### Deploy to gh-pages

Build and deploy to Github Pages:

```
npm run build
npm run deploy
```

## Used packages

- MD to HTML: https://github.com/showdownjs/showdown
- HTML to MD: https://github.com/mixmark-io/turndown
